# Luigi the worm
Gioco endless runner 2D 
## Caratteristiche

- Controlli:
  - `SPAZIO` - Salta
  - `FRECCIA GIÙ` - Scivola
  - `P` - Pausa

- Gameplay:
  - Personaggio che corre automaticamente
  - Ostacoli casuali (alti e bassi)
  - Velocità progressiva
  - Salvataggio miglior punteggio in localStorage

- Grafica:
  - Background effetto parallasse
  - Stile minimal moderno

## Installazione

1. npm install

2. npm run build

3. npm start
